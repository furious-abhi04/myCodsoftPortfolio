PassWord Manager

I developed a secure password manager using the MERN stack, featuring session-based authentication, password hashing, and CRUD operations. The React frontend allows users to manage passwords, while the Node.js/Express backend handles API requests and MongoDB stores the data. Protected routes ensure only authenticated users can access sensitive areas. This project enhanced my skills in React state management, API integration, and implementing security measures.

Pandemic Cartographer

The Pandemic Cartographer is a web-based project that tracks global COVID-19 cases using HTML, CSS, and JavaScript. It integrates the MapTiler SDK to display real-time data on an interactive world map. By fetching live data through a REST API, it visually represents infection hotspots with color-coded markers based on case severity. The project aims to provide an intuitive way to monitor and understand the pandemic's spread.
